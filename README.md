# Personal-Portfolio-Website
Personal portfolio website built with semantic HTML5. It includes About, Skills, and Contact sections with an accessible, validated contact form. Week 1 project for Web Development Internship

Student: Nandini Mourya
Course: B.Tech CSIT — Web Development Internship

# Project Description

This is a single-page personal portfolio website that introduces who I am, lists my technical and soft skills, and provides a way for visitors to contact me. The project was built using HTML5 only — no CSS or JavaScript — with the goal of understanding semantic document structure, accessibility, and native HTML5 form validation.

---

#Pages and Sections

| Section | Description |
|---|---|
| Header | Name, title, welcome message, and profile photo |
| Navigation | Internal anchor links to About, Skills, and Contact |
| About | Biography, education, project highlights, and career goals |
| Skills | Technical skills (MERN stack), soft skills, and hobbies |
| Contact | Form with name, email, phone, and message fields |
| Footer | Copyright, email link, and social media links |

# Key HTML5 Concepts Covered

- Semantic layout tags: `header`, `nav`, `main`, `section`, `footer`
- Proper heading hierarchy (`h1` → `h2` → `h3`)
- Ordered list (`ol`) for ranked technical skills
- Unordered lists (`ul`) for soft skills and hobbies
- Contact form with `required`, `minlength`, `maxlength`, `pattern`, `type="email"`, `type="tel"`
- `label` elements linked to every input via matching `for` and `id` attributes
- Profile image with descriptive `alt` text for accessibility
- Internal anchor navigation using `href="#section-id"`
